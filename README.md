## Feature

- 🌠 국내 최대 포털 네이버의 검색 API 활용 웹페이지   
- 🌠 최신 영화 및 장르 별로 필터링이 가능한 페이지 구성     
- 🌠 미디어 쿼리를 활용한 반응형 지원

## Sitemap
